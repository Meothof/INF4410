-Compiler les sources:
Dans le répertoire ResponseTime_Analyzer: ant

-Pour démarrer le serveur:
Dans le répertoire ResponseTime_Analyzer: ./server

-Créer un fichier:
./client create «nom du fichier»

-Lister les fichiers sur le serveur:
./client list

-Synchroniser le repertoir local avec le serveur:
./client syncLocalDir

-Récupérer un fichier sur le serveur:
./client get «nom du fichier»

-Vérouiller un fichier sur le serveur:
./client lock «nom du fichier»

-Envoyer une nouvelle version d'un fichier sur le serveur:
./client push «nom du fichier»
